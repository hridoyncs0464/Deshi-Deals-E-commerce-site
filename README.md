## ðŸ“˜ Project Overview

**Emergency Hotline** is a **responsive, interactive web application** that allows users to access important emergency service contacts quickly and efficiently.  

The platform enables users to:  
- View emergency services with category, hotline number, and icons  
- Call a service directly and deduct coins for each call  
- Copy hotline numbers to the clipboard  
- Track all called services dynamically in a **Call History section**  

The project was developed as part of **Programming Hero â€“ Batch 12, Assignment 005** and emphasizes **DOM manipulation, event handling, and dynamic content updates using Vanilla JavaScript**.
